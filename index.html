01_PROJECTS

Out-of-Order RISC-V Core

Tomasulo · Split-LSQ · Caches · Verification

Designed and implemented a Tomasulo-style out-of-order RISC-V core with a split load/store queue, parameterized cache hierarchy, and full Spike/RVFI verification. The design emphasizes correctness under aggressive reordering, precise exception handling, and measurable IPC improvements.

SystemVerilog Spike/RVFI Verdi VCS RISC-V

FPGA Pac-Man

AXI4-Lite · BRAM · Sprites · HDMI

Implemented a real-time Pac-Man game on FPGA with HDMI output using a custom RTL graphics pipeline. The design renders a tile-based maze and sprites from BRAM, controlled via an AXI4-Lite interface by C code running on an embedded processor, emphasizing tight hardware–software integration.

Vivado RTL C FPGA

College Dashboard

React · Electron · Productivity · Desktop App

Designed and built a centralized desktop dashboard to help college students manage coursework, tasks, and schedules in one place. The application focuses on clean UX, fast iteration, and extensibility as a real, long-term product rather than a class assignment.

React Electron UI/UX
